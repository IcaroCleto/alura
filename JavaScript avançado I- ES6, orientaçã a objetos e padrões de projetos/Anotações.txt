_propriedade = é uma convenção que indica que essa propriedade só pode ser acessada de dentro da classe.

let é igual a var, com a diferença que o let faz um escopo de bloco, evitando assim que as variaveis vazem para o escopo global.

arrow function = 
1 - Deixa o código menos verboso.
2 - Não é preciso escrever a palavra reservada function.
3 - Quando se tem uma unica instrução numa arrow function vc pode omitir o bloco {}
4 - Já faz o retorno da operação, não precisa usar a palavra reservada return.

Spread operator = Permite passar um array como parametro para uma função, desmembrando o array.
